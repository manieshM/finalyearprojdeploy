# Demo Script

## Demo Flow

1. Open the login page
2. Show separate Admin Login and User Sign In
3. Log in as admin
4. Show recognition dashboard
5. Register a missing person using multiple reference images
6. Use webcam matching
7. Use uploaded image matching
8. Show matched result with image and confidence
9. Show match history
10. Open Case Management page
11. Show lifecycle status update and review queue
12. Open User Access page and show admin approval of case-edit access
13. Show account security option

## Short Explanation

This system is designed to help identify missing persons using face recognition. Users can securely log in, register missing person records with multiple reference images, and then check a live or uploaded image against stored profiles. If a match is found, the system shows the identified person, stores the event in history, and supports case review and lifecycle management.
